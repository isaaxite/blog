快速排序的JavaScript实现

发表于 2020-01-01 更新于 2023-03-08 分类于 数据结构

基本思想

1 在数据集之中，选择一个元素作为”基准”（pivot）。
2 所有小于”基准”的元素，都移到”基准”的左边；所有大于”基准”的元素，都移到”基准”的右边。
3 对”基准”左边和右边的两个子集，不断重复第一步和第二步，直到所有子集只剩下一个元素为止。
