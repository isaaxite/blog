# 描述属性

介绍属性配置详情编写的格式。


## 格式

首行写属性名，可以用md的强调或者标题语法。

空一行，然后分别用两行写该属性的可选项，写默认项。

再空一行，然后编写这个属性的作用，再分述每个可选项的作用。

```
<属性名>

Type：<型类>
Default：<def setting>

<详细描述该属性的作用，分述不同配置的作用>
```
